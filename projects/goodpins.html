Overview

GoodPins is a Chrome extension engineered for developers who demand control over their bookmark workflow. The UI layer is driven entirely by vanilla JavaScript, backed by modular controllers that orchestrate drag-and-drop interactions, theme switching, and state hydration without relying on third-party frameworks. Every interaction is designed to feel native to Chrome thanks to the extension action service worker and a purpose-built popup experience.

Features Overview

Folder Organization

Create and manage nested folders with unlimited depth. Each folder can contain bookmarks and sub-folders, allowing you to organize your links exactly how you want. Folders display the number of items they contain and support custom icons.

GoodPins

Drag & Drop Interface

Effortlessly reorganize your bookmarks and folders with intuitive drag-and-drop functionality. Simply click and drag any item to reorder it. Pinned items are protected from accidental moves, ensuring your important links stay in place.

GoodPins

Keyboard Shortcuts

Boost your productivity with custom keyboard shortcuts. Assign Alt+Shift+1-3 to your most frequently used bookmarks and folders for instant access. Open GoodPins with Alt+Shift+G. Each shortcut displays a badge on the assigned item for easy identification.

GoodPins - Keyboard Shortcuts

Visual Color Tagging

Organize visually with 8 color-coded tags. Each color represents a category: Priority (Yellow), Work (Orange), Personal (Purple), Education (Pink), Tools (Green), Info (Blue), In Progress (Lime), and Other (Gray). Apply tags to both bookmarks and folders for instant visual recognition.

GoodPins - Color Tags

Beautiful Themes

Personalize your experience with 6 carefully crafted themes: Midnight (black), Ocean (blue), Forest (green), Sunset (orange), Lavender (purple), and Rose (pink). Each theme uses CSS variables for instant switching without reloading. Customize grid layout (2, 3, or 4 columns) to match your workflow.

GoodPins - Themes

Pin Important Items

Lock your most important bookmarks and folders in place with the pin feature. Pinned items cannot be dragged or reordered accidentally, ensuring they always stay exactly where you need them. Perfect for frequently accessed links that should remain at the top of your grid.

Engineering Challenges

Recursive Drag & Drop

Problem

Handling drag and drop in a deeply nested tree structure without a library is complex and prone to bugs.

Solution

Implemented a recursive algorithm to calculate drop targets and prevent circular references (dropping a folder into itself).

State Synchronization

Problem

Keeping the Popup UI in sync with chrome.storage changes from other devices or background processes.

Solution

Used event listeners on chrome.storage.onChanged to trigger partial UI updates instead of full re-renders.

DOM Performance

Problem

Rendering thousands of bookmarks caused popup lag and high memory usage.

Solution

Implemented Virtual DOM-like diffing for folder contents and lazy loading for deep subtrees.
