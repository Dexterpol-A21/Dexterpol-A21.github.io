Overview

GoodPins is a Chrome extension engineered for developers who demand control over their bookmark workflow. The UI layer is driven entirely by vanilla JavaScript, backed by modular controllers that orchestrate drag-and-drop interactions, theme switching, and state hydration without relying on third-party frameworks. Every interaction is designed to feel native to Chrome thanks to the extension action service worker and a purpose-built popup experience.

Data is persisted through the chrome.storage API with real-time synchronization between popup sessions, while JSON import/export utilities give power users a versionable configuration file. A dynamic modal system controls layout options (columns, density) and six CSS-variable themes, all applied instantly without reloads. The result is a lightweight yet robust productivity tool that highlights mastery of Chrome APIs, state management, and UI engineering.

Code Highlights

popup.js - Dynamic Rendering & State Management
Core rendering logic that dynamically generates pins and folders from chrome.storage with real-time updates and proper event handling.
/**
 * Load pins and settings from Chrome storage and render them
 */
function loadPins() {
  chrome.storage.sync.get(['pins', 'gridColumns', 'theme', 'shortcuts'], (result) => {
    const pins = result.pins || [];
    gridColumns = result.gridColumns || 3;
    currentTheme = result.theme || 'dark';
    
    pinsGrid.style.setProperty('--grid-columns', gridColumns);
    applyTheme(currentTheme);
    updateGridSizeButtons();
    updateThemeButtons();
    renderPins(pins);
  });
}

/**
 * Render all pins/folders to the grid
 */
function renderPins(pins) {
  // Clear existing items
  document.querySelectorAll('.pin-item, .folder-item').forEach(item => item.remove());

  pins.forEach((pin, index) => {
    if (pin.type === 'folder') {
      renderFolder(pin, index);
    } else {
      renderPin(pin, index);
    }
  });
  
  positionAddMenu();
}
popup.js - Custom Drag & Drop System
Built-from-scratch drag-and-drop implementation using native browser APIs, with pinned item protection and smooth array reordering.
function handleDragStart(e) {
  if (this.classList.contains('pinned')) {
    e.preventDefault();
    return;
  }
  
  draggedElement = this;
  this.classList.add('dragging');
  e.dataTransfer.effectAllowed = 'move';
}

function handleDrop(e) {
  if (e.stopPropagation) e.stopPropagation();
  e.preventDefault();

  if (this.classList.contains('pinned')) {
    this.classList.remove('drag-over');
    return false;
  }

  if (draggedElement !== this) {
    const fromIndex = parseInt(draggedElement.dataset.index);
    const toIndex = parseInt(this.dataset.index);

    chrome.storage.sync.get(['pins'], (result) => {
      const pins = result.pins || [];
      let targetArray = getTargetArray(pins, folderPath);
      
      // Prevent dropping onto pinned items
      if (targetArray[toIndex].pinned) return false;
      
      // Reorder array
      const [movedPin] = targetArray.splice(fromIndex, 1);
      targetArray.splice(toIndex, 0, movedPin);
      
      // Persist changes
      chrome.storage.sync.set({ pins }, () => {
        refreshCurrentView();
      });
    });
  }
  return false;
}
popup.js - Nested Folder Navigation
Hierarchical folder system with unlimited nesting depth, path tracking, and seamless back navigation through folder trees.
/**
 * Get the target array based on folder path
 */
function getTargetArray(pins, path) {
  let targetArray = pins;
  for (let i = 0; i < path.length; i++) {
    if (!targetArray[path[i]].items) {
      targetArray[path[i]].items = [];
    }
    targetArray = targetArray[path[i]].items;
  }
  return targetArray;
}

/**
 * Open a folder and navigate into it
 */
function openFolder(index) {
  chrome.storage.sync.get(['pins'], (result) => {
    const pins = result.pins || [];
    const targetArray = getTargetArray(pins, folderPath);
    const folder = targetArray[index];
    
    if (folder && folder.type === 'folder') {
      folderPath.push(index);
      currentFolder = index;
      currentFolderOpenAllMode = folder.openAll || false;
      
      // Update UI
      folderNav.style.display = 'flex';
      folderNameDisplay.textContent = folder.name;
      openAllToggle.checked = currentFolderOpenAllMode;
      
      renderPins(folder.items || []);
    }
  });
}

Architecture Overview

Popup Layer (popup.js/html)

Controls all UI logic, event listeners, and DOM rendering. Acts as the "brain" of the interface, handling user interactions and orchestrating state updates.

  • Dynamic DOM manipulation with vanilla JS
  • Event delegation for optimal performance
  • Real-time UI updates on storage changes

Background Service Worker

Listens for global events including keyboard shortcuts (chrome.commands) and installation events (chrome.runtime.onInstalled). Manages cross-tab communication.

  • Keyboard shortcut handling
  • Context menu integration
  • Installation & update lifecycle

Storage Layer (chrome.storage.sync)

Acts as the single source of truth for application state. Stores pins, folders, and user settings with asynchronous read/write operations, enabling cross-device synchronization.

  • Asynchronous storage API with sync
  • JSON import/export functionality
  • Persistent and synced across devices
